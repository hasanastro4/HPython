Welcome to HPython's documentation!
===================================

HPython consists of a neccesary modules for simulating a particles or 
particles on smooth potential. This modules hosted in 
`github <https://github.com/hasanastro4/HPython>`_. The main author is Hasanuddin. 
Please email to hasanuddin@physics.untan.ac.id for any questions.
